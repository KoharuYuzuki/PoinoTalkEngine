# PoinoTalk Engine
PoinoTalkの音声合成エンジン  

## インストール
```
# 最新版をインストール
$ bun add github:KoharuYuzuki/PoinoTalkEngine

# タグを指定してインストール
$ bun add github:KoharuYuzuki/PoinoTalkEngine#<tag>
```
